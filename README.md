# Мій Веб-сайт

Ласкаво просимо на мій проект веб-сайту!

## Структура проекту

```
Сайт/
├── index.html          # Головна сторінка
├── css/
│   └── style.css       # Основні стилі
├── js/
│   └── script.js       # JavaScript логіка
├── images/             # Папка для зображень
├── assets/             # Папка для додаткових ресурсів
├── README.md           # Цей файл
└── .gitignore          # Файли для ігнорування Git
```

## Початок роботи

1. Відкрийте `index.html` у вашому браузері
2. Відредагуйте содержимое файлів за необхідністю
3. Додавайте нові сторінки та стилі

## Основні файли

- **index.html** - Головна сторінка з базовою структурою
- **css/style.css** - Стилі для дизайну сайту з адаптивним макетом
- **js/script.js** - JavaScript для інтерактивності
